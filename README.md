# 👗 StyleAI – Smart Fashion Assistant

A modern AI-powered fashion assistant web app that helps users generate outfit ideas, explore trends, and improve their styling decisions based on preferences.

---

## 🚀 Live Demo

🔗 https://mariaprincyaruldass-ops.github.io/fashion-ai-app/

---

## 📌 Features

### 👕 Outfit Recommendation System

* Personalized outfit suggestions based on:

  * Occasion
  * Season
  * Weather
  * Color preference
  * Style type
* Includes:

  * Outfit description
  * Color combinations
  * Styling tips

 ⚡ Quick Outfit Generator

* Instant outfit ideas with:

  * Top wear
  * Bottom wear
  * Footwear
  * Accessories

 📊 Fashion Trends Section

* Explore trending styles:

  * Streetwear
  * Formal wear
  * Seasonal trends
* Includes styling insights and color palettes

 🎨 Color & Style Intelligence

* Smart color combinations
* Basic color psychology suggestions

 ❤️ Favorites & History

* Save your favorite outfits
* View previous suggestions
